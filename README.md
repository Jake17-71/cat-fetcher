# Cat Fetcher

Учебный проект для практики работы с Fetch API, REST API и асинхронными запросами. Приложение позволяет искать изображения котов через API сервиса [cataas.com](https://cataas.com/) (Cat as a Service).

## О проекте

Главная цель проекта — изучение работы с HTTP-запросами, обработки ответов от API и работы с blob-данными. Приложение демонстрирует базовые принципы взаимодействия с REST API и обработку различных состояний запроса (загрузка, успех, ошибка).

## Функциональность

### Поиск по тегам
Форма поиска с автодополнением позволяет выбрать один или несколько тегов для фильтрации изображений. Реализован полноценный UI с:
- Автодополнением при вводе
- Множественным выбором тегов
- Визуальным отображением выбранных тегов
- Возможностью удаления тегов

Запрос формируется в формате: `https://cataas.com/cat/{tags}?type=medium`

### Случайный кот
Кнопка "Мне повезет!" загружает случайное изображение кота без применения фильтров.

Запрос: `https://cataas.com/cat?type=medium`

### Дополнительные возможности
- **Индикатор загрузки** — визуальная обратная связь во время выполнения запроса
- **Обработка ошибок** — различные сообщения для разных типов ошибок (404, network, загрузка изображения)
- **Кнопка прокрутки вверх** — появляется при скролле для быстрого возврата к форме поиска

## Технические детали

- Работа с Fetch API для выполнения HTTP-запросов
- Обработка blob-данных и создание URL для изображений
- Использование промисов для асинхронных операций
- CustomEvent для связи между модулями
- Обработка различных HTTP статус-кодов
